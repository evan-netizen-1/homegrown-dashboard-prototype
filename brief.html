3. Proposed Vendor Dashboard Redesign

Interactive prototype: evan-netizen-1.github.io/homegrown-dashboard-prototype

The Core Insight

The dashboard is designed for an active vendor (orders, quick actions, stats) but 100% of new vendors need onboarding guidance. The product form demands 8 fields upfront when vendors need 3. The most important action (sharing the store link) is buried. Every empty state shows zeros with no guidance.

5 Views in the Prototype

View 1: New Vendor Dashboard

  • Simplified sidebar (4 items vs 9) — removes Posts, Events, Inventory, Finances, Orders during setup
  • Trial countdown banner at top (was hidden — Recording #11 shows vendor hunting for this)
  • Setup checklist replaces tour modal (4 steps, 2 pre-checked for endowed progress effect)
  • Location sequenced BEFORE product (60 seconds vs 7+ minutes — easy win first)
  • Storefront preview inline (was buried in settings dropdown)

Research: Endowed progress effect increases task completion by +30%.

View 2: Product Quick Create

  • Only 3 fields visible: Name, Category, Price (was 8 fields)
  • Everything else collapsed under "Add more details (optional)"
  • "Write it for me" AI button on description field
  • Images marked "Optional now. Required before publishing." (was required upfront — blocks every mobile vendor)
  • No premature validation errors (was showing red errors for untouched fields)
  • SKU moved to bottom, labeled "Optional"

Research: Reducing 11 to 4 fields = +120% form conversions (Imagescape study).

View 3: First Product Celebration

  • Full-screen celebration moment when first product is saved (currently invisible — just returns to product list)
  • Pre-written share templates for Text, Instagram, Copy Link
  • Copy-paste message: "Hey! I just set up an online store. You can order my sourdough anytime: [link]. Order by Friday for Saturday pickup!"
  • Bridges the gap between "product saved" and "first order"

View 4: Post-Setup Dashboard

  • Full sidebar unlocked (all 9 items)
  • Hero card: "Share your store link to get your first order"
  • 3 share methods with pre-written copy: Text Your Regulars, Post on Instagram, Share on Facebook
  • Store link with copy button
  • Orders + Stats below (with empty state messages, not blank)
  • Quick Actions at bottom (relevant now that setup is done)

View 5: Orders Empty State

  • Currently: Completely blank white page
  • Proposed: Package icon + "No orders yet" + "Share your store link to get your first order" + orange CTA + help article link

Research: 84% of users abandon products with empty/blank states (Appcues study).
